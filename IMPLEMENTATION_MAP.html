Boundary Rules

Existing Guidance Stays Authoritative

New analysis paths extend the existing Excel/spreadsheet CRUD guidance; they do not replace it.

No Credential Leakage

Local agents and review packages stay credential-free; Google access authority remains inside approved external access surfaces.

LLM Proposals Need Gates

LLM outputs are candidate claims. Deterministic gates and provenance decide accepted, review-required, blocked, contradicted, or rejected status.

Rollback Is Explicit

Creation and refactor plans return explicit rollback instructions, and write/readback evidence comes from approved external access surfaces.

Excel Values Need Excel

Formula text can be written and scanned deterministically, but formula-result authority requires the real Microsoft Excel engine. Large workbook copy writes also prefer the installed Excel engine to avoid risky full-file round trips.
