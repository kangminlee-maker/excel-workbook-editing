Key Decisions

Credential Boundary

Tokens and service credentials stay in browser auth, local gcloud memory, or Cloud Run broker runtime. Broker policy uses only verified identity, while local agent and artifacts receive sanitized snapshots, plans, and results.

Live Document Safety

Existing Google Sheets are edited in place. No `.xlsx` round-trip is used for connected Sheets.

Apply Plan Gate

Writes require broker identity verification, DWD subject match, policy, schema, operation, risk, approval, precondition, protection, before-state, budget, cancellation, readback, and artifact persistence.

Phase Lock

If no phase is specified, Codex implements Phase 0 only. The file tree is an allowed inventory; phase sections control creation timing.

Protocol Ownership

apply.record requests durable native-host persistence; apply.result is the response with final status and artifact path.

Reference Tools

googleworkspace/cli may guide API exploration and CLI UX, but it is not a runtime dependency, subprocess backend, or credential path.

Domain Evidence Boundary

Domain packs are document-specific evidence, not automatic context. accounting-kr applied only to the earlier accounting workbook sample; the current connected Sheets sample is cash-basis operational reporting and excludes that domain pack.

Adjudicated Claims

LLM and deterministic parser outputs enter as claims. Gates classify status; document graphs, dataflow graphs, ontology graphs, review queues, and LLM context packs are projections over adjudicated claims and evidence.
