# Development Record

This folder contains history-only material from prior implementation paths,
pilot runs, superseded plans, and review evidence.

These records are not active runtime guidance. Runtime code, active design
documents, packaging instructions, MCP tool contracts, and implementation maps
must not depend on or link to these files.

Current groups:

- `historical-docs/archive/`: older process summaries, historical tasklists, and
  superseded design baselines.
- `historical-docs/superseded-docs/`: retired Chrome extension, native-host,
  Claude handoff, and old goal-plan documents.
- `managed-deployment-review-packages/`: historical managed-deployment review
  packages and pilot artifacts.
- `legacy-cleanup/`: diagrams or notes created to explain what was retained,
  archived, or retired during cleanup.
