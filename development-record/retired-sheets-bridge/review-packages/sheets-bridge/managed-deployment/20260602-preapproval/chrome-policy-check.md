# Chrome Policy Check

Run this after Chrome Web Store approval and Admin Console pilot policy
application.

## Steps

1. Open `chrome://policy` in the pilot user's Chrome profile.
2. Click reload policies.
3. Confirm the extension policy includes
   `jahlkdjaokmjbipfhlhnjggcgjmpeiij`.
4. Confirm the update URL is
   `https://clients2.google.com/service/update2/crx`.
5. Open `chrome://extensions`.
6. Confirm `Chrome Sheets Bridge` is installed or installable.
7. Confirm the extension id is
   `jahlkdjaokmjbipfhlhnjggcgjmpeiij`.
8. Confirm the installed version matches the approved Web Store version.
